Work c

IRI https://dcmi.github.io/openwemi/ns#Work
Is Defined By openWEMI Vocabulary
Description

  • An abstract notion of an artistic or intellectual creation.

  • A work is an abstraction that represents the concepts behind a creation. Each metadata community will have a different definition of Work, but it should be the broadest definition of the creations being described.

Expression c

IRI https://dcmi.github.io/openwemi/ns#Expression
Is Defined By openWEMI Vocabulary
Description

  • A perceivable form of the creation.

  • Creations are made tangible through some kind of expression, such as text, sound, or a visual form. There may be more than one expression of a work of the same type or of different types.

Manifestation c

IRI https://dcmi.github.io/openwemi/ns#Manifestation
Is Defined By openWEMI Vocabulary
Description

  • The physical embodiment of a creation.

  • A manifestation is a creation that has been realized in some physical or digital form. A manifestation may be a single realization or a realization that is produced in multiple copies such as the publication of a music compact disk or the manufacture of a shirt.

Item c

Is Defined By openWEMI Vocabulary
Description

  • An exemplar of a creation.

  • An Item is a single instance of the creation, something that can have a place on a shelf or a location online.

Is Defined By openWEMI Vocabulary
Description

  • One responsible for the creation, production, distribution, maintenance, or ownership of a created entity.

  • It is assumed that creations have been affected by an agent with the ability to act. The types of agents is very broad and not constrained by this property.

common Endeavor p

IRI https://dcmi.github.io/openwemi/ns#commonEndeavor
Is Defined By openWEMI Vocabulary
Description

  • Relates any two resources that are or contain the same endeavor.

  • commonEndeavour is intended to define a relationship between two resources that may or may not be modeled as openWEMI. This makes it possible to assert that a resource modeled using a vocabulary that does not use the openWEMI entity model explicitly is describing the same Work, Expression, Manifestation, and/or Item as another resource.
