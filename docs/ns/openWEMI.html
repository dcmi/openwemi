Title

OpenWEMI vocabulary

Creator

Dublin Core Metadata Initiative, OpenWEMI Working Group

Description

OpenWEMI is a minimally constrained vocabulary for describing created resources using the concepts of Work, Expression, Manifestation, Item.

Work c

IRI https://dcmi.github.io/openwemi/ns#Work
Is Defined By OpenWEMI vocabulary
Description

  • A work is an abstraction that represents the concepts behind a creation. Each metadata community will have a different definition of Work, but it should be the broadest definition of the creations being described.

  • An abstract notion of an artistic or intellectual creation.

Manifestation c

IRI https://dcmi.github.io/openwemi/ns#Manifestation
Is Defined By OpenWEMI vocabulary
Description

  • A manifestation is a creation that has been realized in some physical or digital form. A manifestation may be a single realization or a realization that is produced in multiple copies such as the publication of a music compact disk or the manufacture of a shirt.

  • The physical embodiment of a creation.
